# Execute using the example MUT_SDK
### Use root mode and enter service folder
  - sudo -s
  - cd service
### Compilation environment
  - export LD_LIBRARY_PATH=LIB_x86_64
### Execute MUT_SDK
  - ./bin/mcu_menu_ap
